Blog de HTML - Apresentação das tags

O que é HTML

É uma linguagem de marcação usada para estruturar textos na web definindo estruturas da pagina acrescentando informações como textos, imagens, videos, audios e etc.

Referências

O que são tags

As tags são elementos de marcação que definem a estrutura da pagina web da pagina web

Apresentação de algumas tags

  • < h1 > ate < h6 >

    Elementos de cabeçalho que realiza a estrutura do conteudo de uma pagina web Titulos e Subtitulos

  • < p >

    Elemento usado para definir um paragrafo de texto

  • < mark >

    Elemento usado para destacar ou marca um texto na página

  • < small >

    Elemento usado para definir textos menores

  • < i >

    Elemento usado para deixar textos em Italico

  • < u >

    Elemento usado para deixar textos Sublinhado

  • < strong >

    Elemento usado para deixar o texto destacado forte ou em negrito

  • < ol >

    Elemento usado para criar um lista ordenada

  • < ul >

    Elemento usado para criar listas não ordenadas

  • < li >

    Elemento usado para definir itens dentro de uma lista

  • < a >

    Elemento usado para criar links ou âncoras permitindo que usuarios naveguem pelas paginas

  • < hr >

    Elemento usado para inserir linhas horizontais

  • < sub >

    Elemento usado para definir textos subscrito ou seja abaixo da linha do texto normal

  • < sup >

    Elemento usado para definir textos sobrescrito ou seja acima da linha do texto normal

  • < blockquote >

    Elemento usado para indocar trecho ou citações de livros ou qualquer tipo de informação

  • < font >

    Elemento usado para indocar a aparencia do texto

  • < del >

    Elemento usado para marcar o texto que foi exluido

  • < abbr >

    elemento usado para abreviações
